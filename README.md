# Super Sales Pro

進階版桌遊網頁架構，已包含：

- 模組化遊戲架構
- 動畫系統
- 卡牌翻牌特效
- 背景音樂控制
- 自動存檔（localStorage）
- AI 對戰架構
- Firebase 多人連線預留
- PWA 安裝支援
- iPhone 全螢幕模式
- 專業桌遊風格 UI

## GitHub Pages 部署

1. Repository 名稱：
   super-sales-pro

2. 上傳全部檔案

3. Settings → Pages
   Deploy from branch → main → /root

## Firebase

請到 Firebase Console 建立 Web App，
並把 firebase-config.js 中的設定改成你的資料。

## 本地測試

可直接開 index.html，
或使用 VSCode Live Server。
